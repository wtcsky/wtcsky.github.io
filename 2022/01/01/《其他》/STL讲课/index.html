1.什么是STL？

STL（Standard Template Library），即标准模板库，是一个具有工业强度的，高效的C++程序库。

2.STL中六大组件

  1. 容器（Container）

  2. 迭代器（Iterator）

  3. 算法（Algorithm）

  4. 仿函数（Function object）

  5. 迭代适配器（Adaptor）

  6. 空间配制器（allocator）

关于容器（Container）

是一种数据结构，以模板类的方法提供。

  • string
  • vector
  • stack
  • queue
  • priority_queue
  • set
  • map

1 ) string(字符串)

比较起char*的字符串，STL库中的string类不必担心内存是否足够、字符串长度等等。为了在我们的程序中使用string类型，我们必须包含头文件。如下：

2 ) vector

vector是向量类型，可以容纳许多类型的数据。(可以理解为动态数组，是封装好了的类）。进行vector操作前应添加头文件#include <vector>

3 ) stack(栈)

先进后出的数据结构。

4 ) queue(队列)

先进先出

5 ) priority_queue(优先队列)

自动排序的队列

6 ) set

去重排序。

7 ) map

map是STL的一个关联容器，它提供一对一的哈希。

  • 第一个可以称为关键字(key)，每个关键字只能在map中出现一次；
  • 第二个可能称为该关键字的值(value)；

关于算法（Algorithm）

是用来操作容器中的数据的模板函数。例如，STL用sort()来对一个vector中的数据进行排序，函数本身与他们操作的数据的结构和类型无关，因此他们可以在从简单数组到高度复杂容器的任何数据结构上使用；

基本

  • end() 和 begin()
  • empty()
  • size()
  • ……

更多

  • sort() 排序

  • swap() 交换

  • reverse() 翻转
