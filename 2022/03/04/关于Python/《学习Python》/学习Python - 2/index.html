1.断言(assert)

这个在其他的语言里没有见到过，这个的用法大抵和if一样，就是用来判断的，但是如果返回的是 false就会让程序【自我毁灭】。

这个是为了发现潜在的BUG，如果这个数值错误会导致BUG，不如在这里直接把他毁灭。也就是说当要确保某个条件一定要成立的时间可以做一个断言assert。
